This folder contains third-party themes for tkinter.
The theme used for this project was breeze-dark
For more info regarding the theme, refer to: https://github.com/thindil/tkBreeze

Instructions for loading theme:

1. Download and extract theme in this folder
2. Go to config/config.py
3. Change both values TTK_THEME and TTK_THEME_FILE in ZeroConfig class with the theme name (.tcl file) and its path,
respectively
